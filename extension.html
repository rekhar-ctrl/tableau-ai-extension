About Dashboard AI Summarizer

This tool uses Claude AI to automatically read your Tableau dashboard data and generate an executive-ready summary — no manual analysis needed.

Below are the technologies powering it:

⚙️
Google Apps Script
Backend — Tableau auth, Claude API calls, caching, Google Docs export
📊
Tableau REST API v3.21
Fetches dashboard screenshot and CSV data for AI analysis
🤖
Anthropic Claude AI
claude-sonnet-4-6 vision model — reads dashboard image + CSV to generate executive briefings
🌐
GitHub Pages
Hosts the extension UI — required for Tableau's iframe security model
📋
Tableau Extensions API v1.12
Auto-detects dashboard, listens for filter + parameter changes
📄
Google Docs
One-click export of AI summaries as formatted, shareable documents
